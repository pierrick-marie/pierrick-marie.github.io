Projet de recherche : Framework QoCIM (Quality of Context Information Model). Un framework pour simplifier le développement d’applications pour l’Internet des Objets. QoCIM permet de définir, collecter, filter puis exploiter la qualité des informations nécessaires au bon fonctionnement des applications de l'IoT. QoCIM s’intègre le cadre du projet ANR INCOME.
- Technologies : Java, Maven
- Licence : LGPL
- https://github.com/pierrick-marie/QoCIM-Framework
git clone https://github.com/pierrick-marie/QoCIM-Framework.git
